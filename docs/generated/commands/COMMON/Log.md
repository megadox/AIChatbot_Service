# Activity: Log

## Summary
로그 메시지를 Debug창에 표시하는 액티비티

## Metadata
- group: `COMMON`
- script: `Log()`
- pattern: `Log\(`
- theme: `Dark_5`

## Properties
| Name | Type | Default | Options | Description |
|---|---|---|---|---|
| `level` | `string` | `"info"` | - | 표시할 로그의 레벨을 지정합니다.<br/>ex) "Info" |
| `msg` | `string` | `-` | - | 로그의 내용에 해당하는 텍스트를 지정합니다.<br/>ex) "로그 기록" |

## Property Notes
### `level`
표시할 로그의 레벨을 지정합니다.
ex) "Info"

### `msg`
로그의 내용에 해당하는 텍스트를 지정합니다.
ex) "로그 기록"
