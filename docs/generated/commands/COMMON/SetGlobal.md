# Activity: SetGlobal

## Summary
Global변수에 값을 할당하는 액티비티

## Metadata
- group: `COMMON`
- script: `SetGlobal()`
- pattern: `SetGlobal\(`
- theme: `Dark_5`
- prefix: `value`

## Properties
| Name | Type | Default | Options | Description |
|---|---|---|---|---|
| `key` | `string` | `-` | - | Global변수의 이름을 지정합니다.<br/>ex) PcName |
| `value` | `string` | `-` | - | Global변수의 값을 지정합니다.<br/>ex) batem_pc |

## Property Notes
### `key`
Global변수의 이름을 지정합니다.
ex) PcName

### `value`
Global변수의 값을 지정합니다.
ex) batem_pc
