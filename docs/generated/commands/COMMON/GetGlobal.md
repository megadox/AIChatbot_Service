# Activity: GetGlobal

## Summary
SetGlobal액티비티에 저장한 Global변수의 값을 Name필드의 지정된 변수에 저장하는 액티비티

## Metadata
- group: `COMMON`
- script: `GetGlobal()`
- pattern: `GetGlobal\(`
- theme: `Dark_5`
- prefix: `value`

## Properties
| Name | Type | Default | Options | Description |
|---|---|---|---|---|
| `key` | `string` | `-` | - | Global변수의 이름을 지정합니다.<br/>ex) PcName<br/>(SetGlobal에서 설정한 key값과 같아야 한다.) |

## Property Notes
### `key`
Global변수의 이름을 지정합니다.
ex) PcName
(SetGlobal에서 설정한 key값과 같아야 한다.)
