# Activity: MinusCollection

## Summary

## Metadata
- group: `COMMON`
- script: `COLLECTIONS.MinusCollection()`
- pattern: `COLLECTIONS\.MinusCollection\(`
- dependencies: `COLLECTIONS`
- theme: `Dark_5`
- prefix: `collection`

## Properties
| Name | Type | Default | Options | Description |
|---|---|---|---|---|
| `orig` | `string` | `-` | - | 원본 컬렉션을 지정합니다.<br/>ex) Acollection |
| `target` | `string` | `-` | - | 제거할 컬렉션을 지정합니다.<br/>ex) Bcollection |
