# Activity: RenameSheet

## Summary
특정 시트명을 변경할 때 사용하는 액티비티

## Metadata
- group: `EXCEL`
- script: `rename_sheet()`
- pattern: `excel\.rename_sheet\(`
- dependencies: `EXCEL`
- theme: `Accent6_5`

## Properties
| Name | Type | Default | Options | Description |
|---|---|---|---|---|
| `newName` | `string` | `-` | - | 변경할 시트명을 지정합니다.<br/>ex) "Sheet2" |
| `sheetName` | `string` | `-` | - | 대상 시트의 이름을 지정합니다.<br/>ex) "Sheet1" |

## Property Notes
### `newName`
변경할 시트명을 지정합니다.
ex) "Sheet2"

### `sheetName`
대상 시트의 이름을 지정합니다.
ex) "Sheet1"
