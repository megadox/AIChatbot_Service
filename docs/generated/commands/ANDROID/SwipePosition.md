# Activity: SwipePosition

## Summary
안드로이드 디바이스의 특정 좌표 위치에서 다른 좌표 위치로 스와이프하는 액티비티

## Metadata
- group: `ANDROID`
- script: `android_swipe_position()`
- pattern: `device\.android_swipe_position\(`
- dependencies: `ANDROID`
- theme: `Accent6`

## Properties
| Name | Type | Default | Options | Description |
|---|---|---|---|---|
| `duration` | `string` | `500` | `500`, `1000`, `1500`, `2000`, `2500` | 스와이프하는 시간을 지정합니다.<br/>ex) 2000 |
| `endX` | `string` | `-` | - | 끝 x좌표를 지정합니다.<br/>ex) 300 |
| `endY` | `string` | `-` | - | 끝 y좌표를 지정합니다.<br/>ex) 400 |
| `startX` | `string` | `-` | - | 시작 x좌표를 지정합니다.<br/>ex) 100 |
| `startY` | `string` | `-` | - | 시작 y좌표를 지정합니다.<br/>ex) 200 |

## Property Notes
### `duration`
스와이프하는 시간을 지정합니다.
ex) 2000

### `endX`
끝 x좌표를 지정합니다.
ex) 300

### `endY`
끝 y좌표를 지정합니다.
ex) 400

### `startX`
시작 x좌표를 지정합니다.
ex) 100

### `startY`
시작 y좌표를 지정합니다.
ex) 200
