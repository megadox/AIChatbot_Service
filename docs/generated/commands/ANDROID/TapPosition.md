# Activity: TapPosition

## Summary
안드로이드 디바이스의 특정 좌표 위치를 탭하는 액티비티

## Metadata
- group: `ANDROID`
- script: `android_tap_position()`
- pattern: `device\.android_tap_position\(`
- dependencies: `ANDROID`
- theme: `Accent6`

## Properties
| Name | Type | Default | Options | Description |
|---|---|---|---|---|
| `taps` | `string` | `1` | `1`, `2`, `3`, `4`, `5` | 탭할 횟수를 지정합니다.<br/>ex) 2 |
| `x` | `string` | `-` | - | 탭할 x좌표를 지정합니다.<br/>ex) 100 |
| `y` | `string` | `-` | - | 탭할 y좌표를 지정합니다.<br/>ex) 200 |

## Property Notes
### `taps`
탭할 횟수를 지정합니다.
ex) 2

### `x`
탭할 x좌표를 지정합니다.
ex) 100

### `y`
