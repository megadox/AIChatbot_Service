# Activity: KeyEvent

## Summary
안드로이드 디바이스에 키 이벤트를 전달하는 액티비티

## Metadata
- group: `ANDROID`
- script: `android_key_event()`
- pattern: `device\.android_key_event\(`
- dependencies: `ANDROID`
- theme: `Accent6`

## Properties
| Name | Type | Default | Options | Description |
|---|---|---|---|---|
| `keyevent` | `string` | `-` | - | 전달할 키 번호를 지정합니다.(키 번호는 안드로이드 keyEvent를 따른다. 안드로이드 개발자 페이지 및 BATEM 메뉴얼 참조)<br/>ex) 3 (홈 키) |

## Property Notes
### `keyevent`
전달할 키 번호를 지정합니다.(키 번호는 안드로이드 keyEvent를 따른다. 안드로이드 개발자 페이지 및 BATEM 메뉴얼 참조)
ex) 3 (홈 키)
