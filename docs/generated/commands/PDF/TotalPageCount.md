# Activity: TotalPageCount

## Summary
PDF파일의 문서 페이지 개수를 Name필드에 지정된 변수에 저장하는 액티비티

## Metadata
- group: `PDF`
- script: `PDF.page_count()`
- pattern: `PDF\.page_count\(`
- dependencies: `PDF`
- prefix: `count`

## Properties
| Name | Type | Default | Options | Description |
|---|---|---|---|---|
| `filePath` | `string` | `-` | - | 페이지 개수를 추출할 PDF 파일의 경로를 지정합니다.<br/>ex) "C:\\pdf\\sample.pdf" |

## Property Notes
### `filePath`
페이지 개수를 추출할 PDF 파일의 경로를 지정합니다.
ex) "C:\\pdf\\sample.pdf"
