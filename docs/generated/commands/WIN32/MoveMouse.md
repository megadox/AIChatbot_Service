# Activity: MoveMouse

## Summary
윈도우 화면의 좌표에 마우스 이동을 수행하는 액티비티

## Metadata
- group: `WIN32`
- script: `WIN32.move_mouse()`
- pattern: `win32\.move_mouse`
- dependencies: `WIN32`
- theme: `Accent3_5`

## Properties
| Name | Type | Default | Options | Description |
|---|---|---|---|---|
| `x` | `string` | `0` | - | 커서 이동할 x좌표의 위치를 지정합니다.<br/> ex) 500 |
| `y` | `string` | `0` | - | 커서 이동할 y좌표의 위치를 지정합니다.<br/> ex) 700 |

## Property Notes
### `x`
커서 이동할 x좌표의 위치를 지정합니다.

### `y`
