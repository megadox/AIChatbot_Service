# Activity: InputBox

## Summary
사용자 입력을 받을 수 있는 창을 실행시키고 Name 필드에 지정된 변수에 사용자 입력값을 받는 액티비티

## Metadata
- group: `WIN32`
- script: `WIN32.InputBox()`
- pattern: `win32\.InputBox`
- dependencies: `WIN32`
- theme: `Accent3_5`
- prefix: `input`

## Properties
| Name | Type | Default | Options | Description |
|---|---|---|---|---|
| `text` | `string` | `-` | - | 입력 박스에 나타낼 컨텐츠 텍스트를 지정합니다.<br/>ex) "사용할 변수를 입력하세요." |
| `title` | `string` | `-` | - | 입력 박스에 나타낼 타이틀 텍스트를 지정합니다.<br/>ex) "사용자 변수 입력창" |

## Property Notes
### `text`
입력 박스에 나타낼 컨텐츠 텍스트를 지정합니다.
ex) "사용할 변수를 입력하세요."

### `title`
입력 박스에 나타낼 타이틀 텍스트를 지정합니다.
ex) "사용자 변수 입력창"
