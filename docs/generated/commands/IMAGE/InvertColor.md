# Activity: InvertColor

## Summary
특정 이미지 파일을 색반전하여 특정 위치에 저장하는 액티비티

## Metadata
- group: `IMAGE`
- script: `IMAGE.invert()`
- pattern: `IMAGE.\.invert\(`
- dependencies: `IMAGE`

## Properties
| Name | Type | Default | Options | Description |
|---|---|---|---|---|
| `destPath` | `string` | `-` | - | 색반전하여 저장할 파일의 경로를 지정합니다.<br/>ex) "C:\result\sample2.jpg" |
| `srcPath` | `string` | `-` | - | 색반전 대상 원본 파일의 경로를 지정합니다.<br/>ex) "C:\imgs\sample.jpg" |

## Property Notes
### `destPath`
색반전하여 저장할 파일의 경로를 지정합니다.
ex) "C:\result\sample2.jpg"

### `srcPath`
색반전 대상 원본 파일의 경로를 지정합니다.
ex) "C:\imgs\sample.jpg"
