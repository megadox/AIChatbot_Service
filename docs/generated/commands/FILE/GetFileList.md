## Metadata
- group: `FILE`
- script: `FILE.GetFiles()`
- pattern: `= FILE\.GetFiles`
- dependencies: `FILE`
- theme: `Accent5_5`
- prefix: `collection`

## Properties
| Name | Type | Default | Options | Description |
|---|---|---|---|---|
| `path` | `string` | `-` | - | 파일 리스트를 얻고자 하는 폴더의 경로를 지정합니다.<br/>ex) "C:\temp" |

## Property Notes
