# Activity: CopyFile

## Summary
특정 위치의 파일을 복사하여 특정 위치로 붙여넣기하는 액티비티

## Metadata
- group: `FILE`
- script: `FILE.CopyFile()`
- pattern: `FILE\.CopyFile`
- dependencies: `FILE`
- theme: `Accent5_5`

## Properties
| Name | Type | Default | Options | Description |
|---|---|---|---|---|
| `dst` | `string` | `-` | - | 복사한 파일의 저장 경로를 지정합니다.<br/>ex) "C:\temp2\sample2.txt" |
| `src` | `string` | `-` | - | 복사할 원본 파일의 경로를 지정합니다.<br/>ex) "C:\temp\sample.txt" |

## Property Notes
### `dst`
복사한 파일의 저장 경로를 지정합니다.
ex) "C:\temp2\sample2.txt"

### `src`
복사할 원본 파일의 경로를 지정합니다.
ex) "C:\temp\sample.txt"
