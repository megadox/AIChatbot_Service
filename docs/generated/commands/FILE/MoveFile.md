# Activity: MoveFile

## Summary
특정 위치로 파일을 이동하는 액티비티

## Metadata
- group: `FILE`
- script: `FILE.MoveFile()`
- pattern: `FILE\.MoveFile`
- dependencies: `FILE`
- theme: `Accent5_5`

## Properties
| Name | Type | Default | Options | Description |
|---|---|---|---|---|
| `dst` | `string` | `-` | - | 이동한 파일의 저장 경로를 지정합니다.<br/>ex) "C:\temp2\sample2.txt" |
| `src` | `string` | `-` | - | 이동할 원본 파일의 경로를 지정합니다.<br/>ex) "C:\temp\sample.txt" |

## Property Notes
### `dst`
이동한 파일의 저장 경로를 지정합니다.
ex) "C:\temp2\sample2.txt"

### `src`
이동할 원본 파일의 경로를 지정합니다.
ex) "C:\temp\sample.txt"
