# BA-Assist 태스크 업데이트 방법

- Product: BA-Assist
- Version: 2.5.0
- Topic: task-update

## User Intent
사용자가 BA-Assist에 이미 로딩된 태스크나 프로젝트 패키지를 새 파일로 업데이트하는 방법과 제약을 알고 싶어 한다.

## Related Questions
- BA-Assist에서 태스크를 업데이트하려면?
- 이미 등록한 fpk 파일을 교체하려면?
- fpx 패키지도 업데이트할 수 있어?
- Add Task로 업데이트할 때 주의할 점은?
- 패키지 파일명이 다르면 어떻게 돼?

## Short Answer
BA-Assist에 이미 로딩된 태스크를 업데이트하려면 스케줄 에디터에서 `Add Task`를 클릭하고 업데이트할 `fpk` 또는 `fpx` 파일을 선택한 뒤 확인 팝업에서 `YES`를 선택합니다. 소스 확인 결과 업데이트 파일 선택 필터는 `*.fpx`와 `*.fpk`를 모두 허용하지만, 기존 패키지 파일명과 새 파일명이 다르면 업데이트를 막고 파일명을 확인하라고 안내합니다.

## Steps
1. BA-Assist 스케줄 에디터 화면에서 `Add Task`를 클릭한다.
2. 파일 검색창에서 업데이트할 `fpk` 또는 `fpx` 파일을 선택한다.
3. 기존에 로딩된 패키지와 같은 파일명인지 확인한다.
4. 업데이트 확인 팝업이 표시되면 `YES`를 클릭한다.
5. 패키지 설명과 갱신 날짜가 반영되었는지 확인한다.
6. 기존 스케줄이 있다면 실행 경로와 스케줄 상태를 함께 점검한다.

## Notes
- 소스의 `UI/TaskUpdate.cs`는 파일 필터로 `*.fpx;*.fpk`를 허용한다.
- 기존 패키지 파일명과 선택한 파일명이 다르면 업데이트가 차단될 수 있다.
- 패키지 경로가 바뀌면 Windows 작업 스케줄러의 실행 인자도 갱신되어야 한다.

## Example Answer
기존 태스크를 업데이트하려면 `Add Task`에서 새 `fpk` 또는 `fpx` 파일을 선택하고 확인 팝업에서 `YES`를 누릅니다. 다만 기존 패키지와 파일명이 다르면 업데이트가 막힐 수 있으므로 같은 패키지 파일명으로 교체해야 합니다.
