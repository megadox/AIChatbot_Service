# BA-Assist 설치와 라이선스 등록 방법

- Product: BA-Assist
- Version: 2.5.0
- Topic: install-and-license

## User Intent
사용자가 BA-Assist 설치 파일을 어디서 받고, 설치 후 라이선스를 어떻게 등록하는지 알고 싶어 한다.

## Related Questions

## Short Answer
BA-Assist는 `https://download.batem.com`에서 설치 파일 `BAssistSetup.exe`를 내려받아 설치합니다. 설치 후 라이선스 등록 화면에서 `Register License`를 누르고 `License > Create License Info.`의 License Information을 복사해 라이선스 담당자에게 보낸 뒤, 받은 License File을 `Import License File`로 등록합니다.

## Steps
1. `https://download.batem.com`에 접속한다.
2. BA-Assist 다운로드 화면에서 `DOWNLOAD`를 눌러 `BAssistSetup.exe`를 받는다.
3. 설치 파일을 실행하고 설치 폴더를 선택해 설치를 완료한다.
4. 설치 완료 후 라이선스 등록 화면에서 `Register License`를 클릭한다.
5. `License > Create License Info.` 탭에서 License Information을 복사한다.
6. 복사한 License Information과 사용자 정보를 라이선스 담당자에게 전달한다.
7. 받은 License File을 `Import License File`로 선택하고 `Register License`를 눌러 등록한다.

## Notes
- 매뉴얼은 라이선스 담당자 메일 예시로 `license@batem.com`을 제시한다.
- 라이선스 정책이나 담당자 전달 방식은 운영 정책에 따라 달라질 수 있다.
- Help 메뉴에도 License와 Request License 항목이 있다.
