# BA-Assist Refresh, Close, Exit, Tray, Help 메뉴 설명

- Product: BA-Assist
- Version: 2.5.0
- Topic: refresh-close-exit-tray-help

## User Intent
사용자가 BA-Assist의 Refresh, Close, Exit, 시스템 트레이 메뉴, Help 메뉴 차이를 알고 싶어 한다.

## Related Questions
- BA-Assist Refresh는 어떤 기능이야?
- Close와 Exit 차이가 뭐야?
- BA-Assist를 완전히 종료하려면?
- 시스템 트레이 메뉴에는 뭐가 있어?
- Help 메뉴에서 무엇을 할 수 있어?

## Short Answer
`Refresh`는 서버 패키지 리스트와 Local Repository 파일 목록을 갱신합니다. `Close`는 BA-Assist 창만 최소화해 백그라운드 실행을 유지하고, `Exit`는 BA-Assist를 완전히 종료합니다. 완전 종료가 필요하면 시스템 트레이 메뉴의 `Exit`를 사용해야 합니다. Help 메뉴에서는 About, License, Manual, Request License, DB Schema update를 제공합니다.

## Steps
1. 패키지 목록이나 Local Repository 파일 목록을 새로 보려면 `Refresh`를 클릭한다.
2. 화면만 닫고 백그라운드 실행을 유지하려면 `Close`를 사용한다.
3. BA-Assist를 완전히 종료하려면 Main 탭의 `Exit` 또는 시스템 트레이의 `Exit`를 사용한다.
4. 시스템 트레이에서 Task List, Logs, Task Schedule, Setting으로 이동할 수 있다.
5. Help 메뉴에서 버전, 라이선스, 매뉴얼, 라이선스 요청, DB Schema update 기능을 확인한다.

## Notes
- 창 우측 상단 Close 버튼은 Foreground 창을 닫지만 BA-Assist는 백그라운드에서 계속 동작할 수 있다.
- 스케줄 이벤트 처리를 위해 백그라운드 실행이 유지되는 동작이다.
- 완전 종료는 Exit 메뉴를 사용해야 한다.
