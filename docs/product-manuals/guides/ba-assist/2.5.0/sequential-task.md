# BA-Assist Sequential Task 설명

- Product: BA-Assist
- Version: 2.5.0
- Topic: sequential-task

## User Intent
사용자가 BA-Assist의 Sequential Task가 무엇인지, 언제 쓰는지, 어떻게 설정하는지 알고 싶어 한다.

## Related Questions
- Sequential Task 에 대해 설명해줘.
- BA-Assist Sequential 기능이 뭐야?
- 여러 태스크를 순서대로 실행하려면?
- Sequential 태스크 설정 방법을 알려줘.
- Queue와 Sequential Task 차이를 알려줘.

## Short Answer
Sequential Task는 BA-Assist에서 여러 개의 태스크를 지정한 순서대로 연속 실행하도록 설정하는 기능입니다. Main 탭의 `Sequential` 기능으로 설정 창을 열고, `Add Sequential`로 항목을 만든 뒤 순차 실행할 태스크를 원하는 순서대로 Drag & Drop하여 구성합니다.

## Steps
1. BA-Assist Main 탭에서 `Sequential` 기능을 선택한다.
2. Sequential 설정 창에서 `Add Sequential` 버튼을 클릭한다.
3. Sequential Task의 이름과 설명을 입력한다.
4. 순차적으로 실행할 태스크를 원하는 순서대로 Drag & Drop한다.
5. 필요하면 태스크 순서를 변경하거나, `X` 버튼으로 항목을 삭제한다.
6. Queue 기능과 함께 쓰는 경우 제약이 있는지 확인한다.

## Notes
- Sequential Task는 여러 개의 태스크를 순차적으로 실행하는 기능이다.
- 태스크를 추가하거나 삭제할 수 있고, 실행 순서도 변경할 수 있다.
- Queue 기능은 Sequential Task와 함께 사용할 수 없다.
- Queue 기능이 사용되는 동안 Sequential Task는 예정 시간이 되어도 실행되지 않는다.
- 매뉴얼의 관련 위치는 `4-6. Sequential`과 Appendix `Q4. Queue 기능과 Sequential 태스크와 함께 사용할 수 있나요?`이다.
