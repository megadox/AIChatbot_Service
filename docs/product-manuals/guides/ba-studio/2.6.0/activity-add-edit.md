## User Intent

사용자가 Activity를 Task Editor에 추가하거나 이동, 삭제, 붙여넣기 하는 방법을 알고 싶어 한다.

대표 질문:
- Activity 추가 방법을 알려줘.
- 툴박스에서 액티비티를 넣으려면?
- Activity를 이동하거나 삭제하는 방법은?
- Drag & Drop으로 자동화 흐름을 만들려면?

## Short Answer

Activity는 왼쪽 Activities 창에서 검색하거나 그룹별 목록에서 찾은 뒤 Task Editor로 Drag & Drop하여 추가한다. 추가된 Activity는 Flow Editor에서 위치를 이동하거나 삭제하고, Properties에서 세부 값을 설정한다.

## Steps

1. 왼쪽 Activities 창에서 사용할 Activity를 찾는다.
2. 검색창에 Activity 이름이나 기능 키워드를 입력해도 된다.
3. 사용할 Activity를 Task Editor의 원하는 위치로 Drag & Drop한다.
4. Activity가 자동화 흐름에 연결되는지 확인한다.
5. Activity를 선택하고 Properties 창에서 필요한 속성을 입력한다.
6. 위치를 바꾸려면 Activity를 드래그해 이동한다.
7. 삭제하려면 Activity를 선택한 뒤 삭제 기능을 사용한다.
8. Task를 실행해 흐름이 의도대로 동작하는지 확인한다.

## Notes

- Activity는 Work Flow를 구성하는 기본 단위이다.
- Activity 그룹은 WEB, WIN32, EXCEL, FILE, COMMON 등 기능별로 나뉜다.
- Activity의 `Title`, `Description`을 정리하면 흐름을 이해하기 쉽다.
