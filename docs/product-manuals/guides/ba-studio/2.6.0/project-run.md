## User Intent

사용자가 프로젝트 전체를 실행하거나 Start Task 기준으로 프로젝트를 실행하는 방법을 알고 싶어 한다.

대표 질문:
- 프로젝트 실행 방법을 알려줘.
- Start Project는 어떻게 사용해?
- 프로젝트 전체를 실행하려면?
- Start Task 기준으로 실행하려면?

## Short Answer

프로젝트 전체를 실행하려면 프로젝트를 연 뒤 툴바의 `Start Project`를 클릭하거나 Project Tree에서 프로젝트를 우클릭해 `Start Project`를 선택한다. 실행은 설정된 `Start Task`에서 시작된다.

## Steps

1. 실행할 프로젝트를 연다.
2. Project Tree에서 Start Task가 올바르게 설정되어 있는지 확인한다.
3. 툴바에서 `Start Project`를 클릭한다.
4. 또는 Project Tree에서 프로젝트를 우클릭하고 `Start Project`를 선택한다.
5. 실행 로그는 Logs 또는 Debug 창에서 확인한다.
6. 실행을 멈추려면 툴바의 `Stop Task`를 사용한다.

## Notes

- 프로젝트 실행은 `Start Task`에서 시작된다.
- Start Task를 바꾸려면 원하는 Task를 우클릭하고 `Set Start Task`를 선택한다.
- 단일 Task 실행은 `Start Task` 또는 `Run Task`를 사용한다.

## Answer Style

프로젝트 실행과 Task 실행의 차이를 짧게 알려준다.
