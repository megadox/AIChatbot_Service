## User Intent

사용자가 프로젝트 실행 시작 Task를 지정하거나 변경하는 방법을 알고 싶어 한다.

대표 질문:
- Start Task 설정 방법을 알려줘.
- 프로젝트 시작 Task를 바꾸려면?
- Task1 말고 다른 Task부터 실행하고 싶어.
- Set Start Task는 어디서 해?

## Short Answer

Project Tree에서 시작 지점으로 사용할 Task를 우클릭하고 `Set Start Task`를 선택하면 해당 Task가 프로젝트의 시작 Task로 설정된다.

## Steps

1. 프로젝트를 연다.
2. Project Tree에서 시작 지점으로 사용할 Task를 선택한다.
3. 해당 Task를 우클릭한다.
4. 팝업 메뉴에서 `Set Start Task`를 선택한다.
5. 프로젝트 실행 시 선택한 Task부터 시작되는지 확인한다.

## Notes

- Start Task는 프로젝트 실행 시 가장 먼저 실행되는 진입점이다.
- 프로젝트를 처음 생성하면 보통 `Task1`이 Start Task로 설정된다.
- Task 이름을 먼저 정리해두면 시작 지점을 관리하기 쉽다.

## Answer Style

우클릭 위치와 메뉴명을 정확히 안내한다.

Start Task를 바꾸려면 Project Tree에서 시작할 Task를 우클릭한 뒤 `Set Start Task`를 선택합니다. 이후 `Start Project`를 실행하면 선택한 Task부터 프로젝트가 시작됩니다.
