## User Intent

사용자가 현재 Task 또는 프로젝트 안의 특정 Task를 실행하는 방법을 알고 싶어 한다.

대표 질문:
- Task 실행 방법을 알려줘.
- 현재 태스크만 실행하려면?
- Run Task는 어디에 있어?
- 특정 Task를 실행하고 싶어.

## Short Answer

현재 열려 있는 Task는 툴바의 `Start Task`로 실행할 수 있고, Project Tree에서 특정 Task를 우클릭한 뒤 `Run Task`를 선택해 실행할 수도 있다.

## Steps

1. 실행할 Task를 연다.
2. 툴바에서 `Start Task`를 클릭한다.
3. 또는 Project Tree에서 실행할 Task를 우클릭한다.
4. 팝업 메뉴에서 `Run Task`를 선택한다.
5. 실행 중 로그나 결과는 개발자도구의 Logs/Debug 창에서 확인한다.
6. 실행을 중지해야 하면 툴바의 `Stop Task`를 사용한다.

## Notes

- `Start Task`는 현재 활성화된 Task를 실행한다.
- `Run Task`는 Project Tree에서 선택한 Task를 실행할 때 사용한다.
- 프로젝트 전체 실행과 단일 Task 실행은 다르다.
