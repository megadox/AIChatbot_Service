## User Intent

사용자가 Flow Editor 대신 Code Editor에서 코드를 작성하거나 속성 값을 코드 형태로 편집하는 방법을 알고 싶어 한다.

대표 질문:
- Code Editor 사용법을 알려줘.
- Flow 화면에서 Code 화면으로 바꾸려면?
- 속성 값을 코드로 입력하려면?
- 자동완성은 지원돼?

## Short Answer

Task Editor에서 `Code` 버튼을 선택하면 Code Editor로 전환할 수 있다. Code Editor에서는 변수, Activity ID, 사용자 정의 함수 등을 활용해 코드를 작성할 수 있고, 속성 편집 창에서도 코드 형태의 값을 입력할 수 있다.

## Steps

1. Task Editor에서 작업할 Task를 연다.
2. Editor 하단 또는 화면의 `Code` 버튼을 선택한다.
3. Code Editor 화면으로 전환되었는지 확인한다.
4. 필요한 스크립트나 코드를 작성한다.
5. Variables 창에 선언한 변수나 Activity ID를 참조한다.
6. Activity 속성에서 코드 입력이 필요한 경우 편집 버튼을 눌러 Code Editor를 연다.
7. 저장 후 실행 또는 디버그로 동작을 확인한다.

## Notes

- Flow Editor는 Drag & Drop 기반 흐름 작성에 적합하다.
- Code Editor는 복잡한 조건, 계산, 스크립트 작성에 유용하다.
- 변수와 Activity ID 자동완성을 활용하면 입력 오류를 줄일 수 있다.
