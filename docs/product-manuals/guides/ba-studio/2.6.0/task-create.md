## User Intent

사용자가 프로젝트 안에 새 Task를 추가하거나 Task를 만드는 방법을 알고 싶어 한다.

대표 질문:
- 프로젝트에 Task를 추가하는 방법을 알려줘.
- 새 태스크는 어떻게 만들어?
- Project Tree에서 Task 추가는 어디서 해?
- Task1 말고 다른 Task를 추가하고 싶어.

## Short Answer

프로젝트를 연 상태에서 Project Tree의 프로젝트 항목을 우클릭하고 `Add Task`를 선택하면 새 Task를 추가할 수 있다. 생성된 Task는 필요에 따라 `Rename`으로 이름을 바꾸고 자동화 흐름에 맞게 구성한다.

## Steps

1. BA-Studio에서 프로젝트를 생성하거나 기존 프로젝트를 연다.
2. 왼쪽 Project Tree에서 프로젝트 항목을 찾는다.
3. 프로젝트 항목을 우클릭한다.
4. 팝업 메뉴에서 `Add Task`를 선택한다.
5. 새 Task가 Project Tree에 추가되었는지 확인한다.
6. Task 이름을 바꾸려면 Task를 우클릭한 뒤 `Rename`을 선택한다.
7. Task Editor에서 필요한 Activity를 Drag & Drop하여 자동화 흐름을 작성한다.
8. 프로젝트 실행 시작점을 바꿔야 하면 해당 Task를 우클릭하고 `Set Start Task`를 선택한다.

## Notes

- 프로젝트를 처음 만들면 기본 Task인 `Task1`이 자동으로 생성된다.
- 여러 Task를 만들면 업무 단계를 기능별로 나누어 관리할 수 있다.
- Task 이름은 역할이 드러나도록 지정하는 것이 좋다.
- Start Task는 프로젝트 실행 시 가장 먼저 실행되는 Task이다.

## Answer Style

Task 추가 위치와 우클릭 메뉴를 명확히 안내한다. 프로젝트 생성과 Task 추가를 혼동하지 않도록 설명한다.
